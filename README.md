# 🏰 Noah Kingdom System
## 企業向けAI契約書分析プラットフォーム

---

## 🚀 概要

Noah Kingdom Systemは、Azure OpenAIを活用した次世代の契約書分析プラットフォームです。
法務業務の効率化と契約リスクの可視化を実現する革新的なSaaSサービスとして開発中です。

---

## ✨ 主要機能

### 🔍 **CIEL-CONTRACT** (実装完了)
- **AI契約書リスク分析**: Azure OpenAI GPT-4 Turboによる高精度分析
- **契約種別自動判定**: 秘密保持契約、業務委託契約等の自動分類
- **リスクスコア算出**: 0-100点の数値化評価システム
- **多言語レポート**: 日本語HTML、英語PDF、修正案PDFの3形式対応

### 👥 **CIEL-EVAL** (設計完了)
- **人事評価AI支援**: 客観的評価分析システム
- **ギャップ分析**: 自己評価と他者評価の比較機能
- **成長支援**: 個人別改善提案レポート

### ⚡ **CIEL-FLOW** (プロトタイプ完成)
- **Slack完全連携**: リアルタイム通知・承認システム
- **ワークフロー自動化**: 契約承認プロセス効率化
- **進捗可視化**: 処理状況のリアルタイム表示
